# Weather-Station
Weather Station for Raspberry Pi 3 B+

Sensors:

BME 280 Temperature, Pressure, Humidity (I2C)

VEML 6075 UVA, UVB, UV-index (I2C)

PMS7003 PM1, PM2.5, PM10 (UART)

SparkFun  SEN-08942

Data send Weather Underground and Időkép
